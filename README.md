# netflix-data-analysis
## 📌 Project Overview
This project analyzes a Netflix dataset to discover insights about movies and shows, such as trends, genres, ratings, and popularity.

## 📂 Dataset
- File Name: `mymoviedb.csv`
- The dataset contains information about movies/shows like title, genre, rating, release year, etc.

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Genre-wise analysis
- Rating distribution
- Year-wise content growth
- Popular movies/shows analysis

## 📁 Project Files
- `netflix_analysis.ipynb` → Main analysis notebook
- `mymoviedb.csv` → Dataset file
- `README.md` → Project documentation

## 🚀 How to Run the Project
1. Download or clone this repository
2. Open the notebook in Jupyter Notebook
3. Run all cells to see analysis and visualizations

## 📈 Results
The project provides insights into:
- Most common genres
- Trends over the years
- Distribution of ratings
